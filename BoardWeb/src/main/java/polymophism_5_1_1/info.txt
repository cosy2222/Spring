	  객체 생성 어노테이션 (@Component)
	  의존성 주입 어노테이션 (DI) 
	  	1. @Autowired  : 주로 변수 위에 설정 해당타입의 객체를 찾아서 주입  (Spring)
	  	2. @Qualifier  : 특정 객체의 이름을 이용해서 의존성 주입할떄 사용   (Spring)
	  	3. @Resource   : @Autowired 와 Qualifier 를 합친기능 (java)
	  	3. @Inject 	   : @Autowired 와 동일한 기능 ( java )
